# 艺创AI-聊天绘画系统「PHP源码版」- UniApp端详细文档

## 项目概述

艺创AI是一款基于UniApp开发的跨平台AI聊天绘画系统，支持微信小程序、H5、APP等多端部署。系统集成了AI聊天对话、AI绘画创作、用户管理、支付系统、任务中心等完整功能模块。

## 技术架构

### 核心技术栈
- **前端框架**: UniApp (Vue3 + TypeScript)
- **状态管理**: Pinia
- **路由管理**: uniapp-router-next
- **UI组件库**: uView Plus (vk-uview-ui)
- **样式方案**: TailwindCSS + SCSS
- **构建工具**: Vite
- **代码规范**: ESLint + Prettier

### 平台支持
- ✅ 微信小程序 (mp-weixin)
- ✅ H5网页 (h5)
- ✅ APP (Android/iOS)
- ✅ 支付宝小程序 (mp-alipay)
- ✅ 百度小程序 (mp-baidu)
- ✅ 抖音小程序 (mp-toutiao)
- ✅ QQ小程序 (mp-qq)

## 功能模块详解

### 1. AI聊天系统

#### 核心功能
- **智能对话**: 支持多种AI模型的智能对话

- **语音输入**: 支持语音转文字输入
- **语音播报**: 支持AI回复语音播报
- **收藏管理**: 支持对话记录收藏
- **分类管理**: 支持对话分类管理

#### 技术实现
- 使用EventStream实现流式响应
- 集成Web Speech API实现语音识别
- 支持Markdown格式渲染
- 支持代码高亮显示

#### 主要API接口
```typescript
// 聊天相关接口
```
